# Untitled Project 1

WebAR Experience.

## Mode: Marker-based
This experience tracks an image target.

## Requirements
- Camera permissions.
- HTTPS hosting.

## ⚠️ Compile Your Marker
Upload `assets/marker_marker.jpg` to [MindAR Image Compiler](https://hiukim.github.io/mind-ar-js-doc/tools/compile) and save result as `assets/targets.mind`.
